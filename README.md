# 🎫 会场AI助手

一个基于活动信息的智能问答助手，支持查询活动的时间、地点、票务、阵容、参展信息等内容。

本项目采用 **轻量 Web 架构 + JSON 数据源**，无需数据库，适合快速部署和演示。

由于本项目的使用教程较为繁琐：需要在Windows系统的提示命令符中使用 “cd”指令打开拉取的代码的文件，再用 “streamlit run”指令打开本网站；
因此在本产品的ai接入后需要重新对于本产品的web开发进行进一步优化。
另外就是关于本产品的依赖软件库，本项目中的 “requestments.txt” 相对来讲是比较混乱的，后续还需更新。

---

## ✨ 功能介绍

- 🔍 自动识别用户提问中的活动
- 🧠 基于结构化数据的精准回答（不编造）
- 💬 支持连续对话（上下文记忆）
- 📁 活动信息以 JSON 文件形式管理
- 🌐 Web 界面（基于 Streamlit）
- 新更新可供主办方将活动信息上传至该网站
