# 会场AI小助手网页版

## 本产品已经正式投入使用，现版本 *v1.0*

### 在线体验：网站地址https://vvp9bxbdffh7gnjsskx9hp.streamlit.app/


## 项目简介：

一个基于活动信息的智能问答助手，支持查询活动的时间、地点、票务、阵容、参展信息等内容。

本项目旨在解决由于主办方等消息的不醒目、接收消息不及时等问题，根据主办方对举办活动的主要信息提交至本网站后，可供用户进行

本项目采用 **轻量 Web 架构 + JSON 数据源**，无需数据库，适合快速部署和演示。

---

## ✨ 功能介绍：

- 🔍 自动识别用户提问中的活动
- 🧠 基于结构化数据的精准回答（不编造）
- 💬 支持连续对话（上下文记忆）
- 📁 活动信息以 JSON 文件形式管理
- 🌐 Web 界面（基于 Streamlit）
- 可供主办方将活动信息上传至该网站

## 功能说明：

- 当前版本支持基于主办方提供的活动信息进行问答
- 系统不会编造未提供的信息
- AI 仅用于回答润色，不参与数据判断

  
## 运行方式：

- 本项目可采取两种使用方式：1. 在线使用，可点击上方链接；2. 本地运行，将代码拉取到本地在python环境中运行。

## 技术架构：
用户问题
   ↓
活动检索
   ↓
语义抽取
   ↓
已知信息
   ↓
AI 润色（可选）
   ↓
回答输出
